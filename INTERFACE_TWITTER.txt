✏️  ÉDITION DE TWEETS
═══════════════════════════════════════════════════════════════

1. Clic sur le bouton "✎" (crayon)
2. Le texte devient éditable
3. Modifiez le texte
4. Cliquez [Sauvegarder]
5. Le tweet est mis à jour en base !

Alternative: Cliquez [Annuler] pour restaurer


🗑️  SUPPRESSION DE TWEETS
═══════════════════════════════════════════════════════════════

1. Clic sur le bouton "🗑" (poubelle)
2. Confirmation demandée
3. Cliquez [OK]
4. Le tweet disparaît avec animation
5. Supprimé de la base de données


🎯 BOUTONS DISPONIBLES
═══════════════════════════════════════════════════════════════

[+]  → Sélectionner le tweet
[✓]  → Tweet déjà sélectionné
[✎]  → Éditer le texte
[🗑] → Supprimer définitivement

🎯 WORKFLOW TYPIQUE
═══════════════════════════════════════════════════════════════

1. Ouvrir http://localhost:5000

2. Cliquer sur un compte (@DearS_o_n par exemple)

3. VOIR LES IMAGES dans les tweets ! ✅

4. Filtrer par type "Original" et "Avec média"

5. Parcourir et voir les images

6. Éditer un texte si besoin (✎)
   → Modifier le texte
   → [Sauvegarder]

7. Supprimer un tweet indésirable (🗑)
   → Confirmation
   → [OK]

8. Sélectionner les tweets à poster (+)
   → Ou "✓ Tout sélectionner"

9. Aller dans Configuration

10. Activer la programmation automatique

11. Les tweets seront postés avec leurs images ! 🎉


🐛 SI LES IMAGES NE S'AFFICHENT PAS
═══════════════════════════════════════════════════════════════

1. Vérifiez que Flask est bien redémarré

2. Rechargez avec Ctrl + Shift + R

3. Ouvrez la console (F12) → Onglet Console
   → Vérifiez s'il y a des erreurs

4. Ouvrez la console (F12) → Onglet Network
   → Rechargez la page
   → Cliquez sur une requête "/media/..."
   → Vérifiez le statut (doit être 200)

5. Vérifiez les logs dans le terminal Flask

6. Vérifiez que les chemins dans le CSV commencent par:
   software_clean/scraped data/.../
