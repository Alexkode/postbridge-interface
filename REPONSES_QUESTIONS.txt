📍 EMPLACEMENT:
/home/cytech/Projects/scraperX/scraper/postbridge_interface/postbridge.db

📋 TABLE: tweet_edits

Structure:
┌──────────────────────────────────────────────────────────┐
│ • account_name   → Nom du compte (@jaynitx, etc.)        │
│ • tweet_id       → ID unique du tweet                    │
│ • original_text  → Texte original                        │
│ • edited_text    → Votre modification                    │
│ • created_at     → Date de création                      │
│ • updated_at     → Date de mise à jour                   │
└──────────────────────────────────────────────────────────┘


🔄 FONCTIONNEMENT:

1. Vous modifiez un tweet
2. Vous cliquez "Sauvegarder"
3. ✅ Enregistré dans postbridge.db
4. ✅ Persistant (même après redémarrage)
5. ✅ Automatiquement rechargé lors de l'affichage


💾 PERSISTANCE:

✅ Vos modifications restent même si:
   • Vous rechargez la page
   • Vous fermez le navigateur
   • Vous redémarrez Flask
   • Vous redémarrez l'ordinateur

❌ Vous perdez vos modifications SEULEMENT si:
   • Vous supprimez postbridge.db
   • Vous réinitialisez la base de données


🔍 VÉRIFIER VOS MODIFICATIONS:

Méthode 1 - Script automatique:
──────────────────────────────────────────────────────────────
cd /home/cytech/Projects/scraperX/scraper/postbridge_interface
./verifier_modifications.sh

→ Affiche un menu interactif avec:
   • Statistiques
   • Liste des modifications
   • Export CSV
   • Recherche par compte


Méthode 2 - Commande SQL directe:
──────────────────────────────────────────────────────────────
# Voir toutes les modifications
sqlite3 postbridge_interface/postbridge.db "SELECT * FROM tweet_edits;"

# Voir les modifications d'un compte
sqlite3 postbridge_interface/postbridge.db \
  "SELECT * FROM tweet_edits WHERE account_name='jaynitx';"

# Compter le nombre de modifications
sqlite3 postbridge_interface/postbridge.db \
  "SELECT COUNT(*) FROM tweet_edits;"


Méthode 3 - Interface graphique:
──────────────────────────────────────────────────────────────
# Installer DB Browser
sudo apt install sqlitebrowser

# Ouvrir la base
sqlitebrowser postbridge_interface/postbridge.db

⚠️ SAUVEGARDE RECOMMANDÉE:

Pour ne jamais perdre vos modifications:

# Sauvegarde manuelle
cp postbridge_interface/postbridge.db \
   postbridge_interface/postbridge.db.backup

# Sauvegarde datée
cp postbridge_interface/postbridge.db \
   ~/backups/postbridge_$(date +%Y%m%d).db

# Automatiser avec cron (optionnel)
0 0 * * * cp ~/Projects/scraperX/scraper/postbridge_interface/postbridge.db \
          ~/backups/postbridge_$(date +\%Y\%m\%d).db
